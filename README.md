### Overleaf HotKeys

```
|------------------|----------------------|---|-------------------|--------------------|
| Command          |  Function            |   |  Command          |  Function          |
|------------------|----------------------|---|-------------------|--------------------|
| Ctrl + Enter     |  Compile             |   |  Ctrl + Y         |  Redo              |
|------------------|----------------------|---|-------------------|--------------------|
| Ctrl + Shift + L |  Go To Line          |   |  Ctrl + /         |  Toggle Comment    |
|------------------|----------------------|---|-------------------|--------------------|
| Ctrl + D         |  Delete Current Line |   |  Ctrl + B         |  Bold text         |
|------------------|----------------------|---|-------------------|--------------------|
| Ctrl + U         |  To Uppercase        |   |  Ctrl + Shift + U |  To Lowercase      |
|------------------|----------------------|---|-------------------|--------------------|
| Ctrl + Space     |  Autocomplete Menu   |   |  Ctrl + Space     |  Search References |
|------------------|----------------------|---|-------------------|--------------------|
| Ctrl + J         |  Toggle review panel |   |  Ctrl + Shift + C |  Add a comment     |
|------------------|----------------------|---|-------------------|--------------------|
```
